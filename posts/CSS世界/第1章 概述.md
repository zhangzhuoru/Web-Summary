# 第1章 概述 #

## CSS3 ##

1. 布局更丰富
    - 移动端媒体查询以及响应式布局，如图片元素的`srcset`属性、`object-fit`属性
    - 弹性盒子布局`flex`
    - 栅格布局`grid`
2. 视觉表现
    - 圆角、阴影和渐变让元素更有质感
    - `transform`变换
    - `filter`滤镜和混合模式
    - `animation`动画

# 第2章 术语和概念 #

1. 属性
2. 值
    - 整数值
    - 数值
    - 百分比
    - 长度值
    - 颜色值
3. 关键字
4. 变量
5. 长度单位
    1. 相对长度单位
        - 相对字体长度单位，如`em`、`ex`
        - 相对视区长度单位，如`vw`、`vh`、`vmin`、`vmax`
    2. 绝对长度单位，如`px`
6. 功能符，值以函数的形式指定，主要用来表示颜色`rgba`、背景图片地址`url`、元素属性值、计算`calc`和过渡效果
7. 属性值，属性冒号`:`后面的所有内容统称为属性值
8. 声明
9. 生命块
10. 规则和规则集
11. 选择器
    - 类选择器`.`
    - ID选择器`#`
    - 属性选择器`[title]{}`、`[title="css"]{}`
    - 伪类选择器`:`
    - 伪元素选择器`::`
12. 关系选择器
    - 后代选择器，**空格**
    - 相邻后代选择器，`>`
    - 兄弟选择器，`~`
    - 相邻兄弟选择器，`+`
13. @规则，`@media`、`@font-face`、`@page`、`@support`

## 未定义行为 ##

未定义行为就是不同浏览器对样式的支持度不一样，得到的效果也不一致。
